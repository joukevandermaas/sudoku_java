Run SudokuApp in an applet viewer to run the application. The
working directory must contain a file sudokus.txt that holds sudokus
in the following format:

79.....23
..38..4..
1..2.6...
6...8.94.
....2....
.31.6...8
...7.1..5
..5..21..

And have UTF-16 encoding. Lines starting with % will be ignored.

For an overview of important classes and methods, see the file
design_overview.pdf.
